# 🛍️ Retail Sales Analysis Dashboard (Power BI)

## 📊 Overview
This project presents a comprehensive Retail Sales Analysis using Power BI.  
The dashboard provides insights into sales performance, customer behavior, and product trends to support business decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-performing products and categories
- Understand regional sales distribution
- Track profit and revenue trends
- Provide actionable business insights

---

## 🛠️ Tools & Technologies
- Power BI


---

## 📈 Key Insights
- Identified top-selling product categories contributing to revenue
- Highlighted high-performing regions and underperforming areas
- Analyzed monthly/quarterly sales trends
- Found key profit drivers and loss-making segments
